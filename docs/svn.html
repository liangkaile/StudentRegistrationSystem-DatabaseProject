1.1 Create hosting project

http://code.google.com or http://developers.google.com

1.2 Customize the project

1.3 Upload project/code from local machine 

svn import [PATH] URL

If PATH is empty then "." is assumed. For example, 

svn import . https://PROJECT-NAME.googlecode.com/svn/trunk

then input the username and password which can be found on http://code.google.com -> profile -> settings
